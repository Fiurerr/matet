Cómo ejecutar el programa
1. Abre un terminal o Jupyter Notebook.
2. Coloca el archivo ScorpionCover.jpg en la misma carpeta que tu script.
3. Ejecuta el script.
4. El programa:
• Mostrará tres imágenes: original, recuperada, y con menor correlación.
• Imprimirá el número de iteraciones necesarias para recuperar la imagen y dónde se alcanzó la menor correlación.
